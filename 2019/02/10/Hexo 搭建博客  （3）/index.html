Hexo 本地配置及优化

1. 博客作者信息设置

# Site
title: Time-C
subtitle: 尽小者大 慎微者著
description:
keywords:
author: Time-fairy
language: zh-Hans
timezone: PRC

# URL
url: https://time-fairy.github.io
root: /
permalink: :year/:month/:day/:title/
permalink_defaults:

2. 添加标签页面

/*
必看！！！！
    确认站点配置文件_config.yml中有： tag_dir： tags （有注释的去掉注释）
    确认主题配置文件_config.yml中有： tags: /tags
*/
hexo new page "tags"

复制代码替换站点中 source/tags/index.md 文件内容：

---
title: tags
date: 2019-02-10 13:04:00
type: "tags"

---

如果站点启用多说或者Disqus，会在默认页面添加评论，关闭评论那么还需要添加:

comments： false

3. 添加标签页面

/*
确认站点配置文件里有 category_dir: categories
确认主题配置文件里有 categories: /categories
*/
hexo new page "categories"

复制代码替换站点的 source/categories/index.md 文件内容：

---
title: categories
date: 2015-10-20 06:49:50
type: "categories"

---

如果站点启用多说或者Disqus，会在默认页面添加评论，关闭评论那么还需要添加:

comments： false

4. 添加自定义页面

hexo new page "title"

如果你不想在该页面显示评论，那么我们需要打开 blog\source\logs\index.md 在date的 下一行添加 comments: false(注意冒号后面有一个空格)

找到 \next_config.yml 下的 menu ， 把 title 加进去。

然后在下面 menu_icons 中加入你想让其显示的图标图标传送

在 /themes/hexo-theme-next/languages/zh-Hans.yml 文件中（这里默认你使用的是简体中文，若是其他语言更改相应的yml就行），在 memu 下加一句即可：

title: 标题
